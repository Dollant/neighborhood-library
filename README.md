# Neighborhood Library

A Java console application that allows users to check books in and out
of a neighborhood library using the honor system.

## Description
The library is a free service open to anyone in the neighborhood.
Anyone can check out a book by simply entering their name.
The application tracks who has each book checked out.

## Features
- View all available books
- Check out a book by entering your name
- View all currently checked out books
- Check in a book using its ID
- Exit the application

## How to Run
1. Clone this repository
2. Open the project in IntelliJ IDEA
3. Navigate to Main.java
4. Click the Run button or press Shift + F10

## Project Structure
- `Book.java` - Book class with properties, getters, setters, and methods
- `Main.java` - Main application file with menu and user interaction
